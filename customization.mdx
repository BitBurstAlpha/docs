title: "Customization"
description: "When using the [low-code](#low-code) approach, you can configure the Chatbot's behavior by adding attributes to the `ab-chat` element or by giving the React component props.
"
---

When you add an attribute to the `ab-chat` or `Chat` element, this value will **override** the value in the Chatbot settings.

See all the available attributes in the [Chatbot Reference](#chatbot-reference).
